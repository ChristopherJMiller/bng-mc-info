How to get started

BNG Minecraft is equipped with plugins to customize the usual vanilla experience! The following is a guide for how to get starting working with these plugins and commands for those not familiar:

Build your home using a Town!

By default, you cannot build or break in the wilderness (with the exception of ores and plants). Claim ownership over a region of the world by forming a town! Towns provide block protection against anyone not in your town and allows you to manage plots for your townmembers!

Quick guide to forming a town.


# Create your town (your current chunk will become the home of the town)
/town new {townname}
# Invite friends to your town!
/town add {playername}
# Claim new chunks for your town
/town claim

Create Warps to Connect the World

Players can create warp locations to connect the world! All warps are public when created and must be warped to either via a command or warp sign. The /to command can only be used within spawn. However, warp signs can be created anywhere given a much higher cost! 


# Create a warp
/warp create {name}
# Warp via command from spawn
/to {warp name}
# View list of all warps
/warp list
# Use a compass to point to a warp location
/warp point {warp name}

See the command reference for more commands. Note that some are unaccessible due to server permissions.
